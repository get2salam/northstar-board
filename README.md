# Northstar Board

A local-first constellation planner. Turn goals and milestones into stars you can
steer by.

Work in progress — this commit establishes the scaffold. The app, design system,
and interactions arrive in later commits.

## Goals

- 100% static: HTML, CSS, vanilla JavaScript. No build step, no backend.
- Local-first: every board lives in your browser via `localStorage`.
- Visual: goals become stars, dependencies become paths, progress shines.
- Portable: import/export the whole board as a single JSON file.
